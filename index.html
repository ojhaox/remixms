Installation Guide:

  1. Download all three files: V83MSSETUP, RemixMS Client, and WZ Files
  2. Run and install the V83MSSETUP
  3. Navigate to C:\Nexon\MapleStory and delete:
    • HShield folder
    • Patcher.exe
    • MapleStory.exe
  4. Extract the WZ Files and copy them into the MapleStory folder
  5. Copy the RemixMS Client into the MapleStory folder

Important: You must disable your firewall, antivirus, and real-time protection before playing. This is a common requirement for all MapleStory private servers.
